A modern full-stack web portal tracking the FIFA World Cup 2026 (hosted across USA, Canada, and Mexico). Built using the cutting-edge **TanStack Start** meta-framework, **React 19**, and **Tailwind CSS v4** to deliver an interactive user experience for tournament schedules, host stadiums, injuries tracker, and team insights.

---

## ⚡ Features

- **Interactive Matches Tracker**: Follow upcoming schedules, live scores, and group stage standings.
- **Qualified Teams Directory**: Dedicated pages detailing squads, statistics, and historical performance (dynamic routing via `teams/$teamId`).
- **Interactive Host Venue Map**: Visualized map pinpointing matches, stadiums, and host city data across North America.
- **Injury & Availability Log**: Live board tracking suspended, injured, and unavailable key players per national squad.
- **Cutting-edge Tech Stack**: Utilizes TanStack Router for type-safe routing, Tailwind CSS v4, and React 19 concurrent features.

---

## 🛠️ Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) (Server-side rendering, type-safe APIs, and loaders)
- **Frontend**: [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **State & Data**: TanStack React Query (data fetching and client caching)
- **Styling**: Tailwind CSS v4 (using the official `@tailwindcss/vite` plugin)
- **UI Components**: Shadcn UI & Radix UI primitives, Lucide Icons, Recharts

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js 18+ and `bun` (or npm/pnpm/yarn) installed.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chaoui-ahmed/world-cup-hub.git
   cd world-cup-hub
   ```

2. **Install dependencies**:
   ```bash
   bun install
   ```

3. **Run in Development mode**:
   ```bash
   bun dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the portal.
